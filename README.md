# Express Template

## 介绍

一个基于 Express 的模板。

## 功能

- 采用 ES Module 语法
- RESTful CRUD API 示例
- 支持解析 HTTP request body，支持如下：
  - `Content-Type: application/json`
  - `Content-Type: application/x-www-form-urlencoded`
  - `Content-Type: multipart/form-data`
  - ...
- 支持解析 Cookies
- CORS 跨域处理
- logger 日志集成
- 对常见错误（400、404、500）进行统一处理
- 静态服务器
- 支持多环境的变量控制
